
# Http server

A simple Http-server with javascript using net module. 


## How to Use






+ clone repo
+ open main.js 
+ add static file in pubilc (or add custom file in app.static('_yourFileName_'))
+ run main.js

## Changing routes

+ Add your own routes___
+  app.route("GET", "_yourRoute_", _yourRouteHandler_);
